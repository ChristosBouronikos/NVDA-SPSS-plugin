# Security Policy

## Supported Versions

Only the latest released version of this add-on is supported for security fixes.

## Reporting A Security Issue

Do not open a public issue for a vulnerability.

Send a private report to:

```text
chrisbouronikos@gmail.com
```

Include:

- Add-on version.
- NVDA version.
- Windows version.
- IBM SPSS Statistics version.
- Clear reproduction steps.
- Expected and actual behavior.
- Any relevant logs.

The maintainer will review the report and prepare a fixed release if the issue is valid.
