# Αλλαγές

## 1.1.1

Έκδοση συντήρησης μετά από σχόλια αξιολόγησης του NVDA Add-on Store.

- Ενημερώθηκε η τελευταία δοκιμασμένη έκδοση NVDA σε 2026.1.1 μετά από δοκιμές με NVDA 2026.1.1 (64-bit), IBM SPSS Statistics 31.0.0.0 (build 117) και Windows 11.
- Αφαιρέθηκαν τα περιττά αρχεία `__init__.py` από τα `appModules` και `globalPlugins`.
- Τεκμηριώθηκε κάθε καταχωρημένο όνομα εκτελέσιμου SPSS με σχόλιο στον κώδικα που εξηγεί ποια έκδοση ή στοιχείο του SPSS το χρησιμοποιεί.
- Αφαιρέθηκαν καταχωρήσεις που έμοιαζαν με τίτλους παραθύρων και μη τεκμηριωμένα aliases εκτελέσιμων (`spssstatistics`, `ibmspssstatistics`).
- Προστέθηκαν ενότητες Δοκιμών και Δήλωσης Χρήσης Τεχνητής Νοημοσύνης στο README στα Αγγλικά και στα Ελληνικά.

## 1.1.0

- Προστέθηκε αναγνώριση, βοήθεια και συντομεύσεις για Overview και Chart Builder.
- Επεκτάθηκαν οι περιγραφές μενού SPSS με καθοδήγηση ανάλογα με Data Editor, Output Viewer, Syntax Editor, Chart Builder και γραμμή μενού.
- Προστέθηκε καθοδήγηση προσβασιμότητας αποτελεσμάτων, συμπεριλαμβανομένων των ρυθμίσεων screen reader για pivot tables και ροών εξαγωγής για μη προσβάσιμα charts, trees και model views.
- Προστέθηκαν ανακοινώσεις τύπου αποτελέσματος, επιπέδου περιγράμματος, θέσης στοιχείου και expanded/collapsed κατάστασης όταν το SPSS τις εκθέτει.
- Προστέθηκαν υποδείξεις ενεργειών στο Variable View για κελιά Type, Values, Missing, Measure και Role.
- Προστέθηκαν ετικέτες για Value Labels, Missing Values, Variable Type, οδηγούς εισαγωγής, διαλόγους επιλογής περιπτώσεων, συνηθισμένους διαλόγους αναλύσεων και περιοχές Chart Builder.
- Προστέθηκε εναλλαγή αναλυτικής/σύντομης καθοδήγησης και εντολή αντιμετώπισης προβλημάτων που γράφει το τρέχον αντικείμενο προσβασιμότητας SPSS στο log του NVDA.
- Επεκτάθηκαν οι ελληνικές μεταφράσεις και η ελληνική τεκμηρίωση για τις νέες ροές εργασίας.
- Προστέθηκε η συντόμευση `NVDA+Control+Alt+T` για ανάγνωση τρέχουσας καρτέλας και πλαισίου πίνακα.
- Βελτιώθηκε η ανάγνωση κελιών τύπου Excel για Data View, Variable View και πίνακες αποτελεσμάτων.
- Προστέθηκαν αναφορές κελιών τύπου A1 όταν το SPSS εκθέτει αριθμό γραμμής και στήλης.
- Προστέθηκε αναφορά κεφαλίδας γραμμής, κεφαλίδας στήλης, ονόματος πίνακα και επιλεγμένης καρτέλας.
- Προστέθηκαν ελληνικά αρχεία μετάφρασης για τα μηνύματα του NVDA.
- Προστέθηκε υποστήριξη ελληνικής τεκμηρίωσης στο πακέτο.
- Προστέθηκαν εντολές για πού βρίσκομαι, διαδρομή μενού, βοήθεια διαλόγου, λίστες διαλόγου, σύνοψη πίνακα, περίγραμμα αποτελεσμάτων, γραμμή/στήλη αποτελεσμάτων, αντιγραφή σύνοψης, σύνοψη γραμμής μεταβλητής και δίγλωσσο γλωσσάρι.
- Προστέθηκαν προαιρετικές αυτόματες ανακοινώσεις κελιών κατά τη μετακίνηση σε πίνακες SPSS.
- Προστέθηκε αναφορά ετικέτας τιμής σε κελιά δεδομένων όταν το SPSS εκθέτει ξεχωριστή ετικέτα από την ακατέργαστη τιμή.
- Συγχωνεύτηκε η τεκμηρίωση έργου σε ένα README με ελληνικές οδηγίες πρώτα και αγγλικές μετά.
- Επεκτάθηκε η κάλυψη μενού με αναλυτικό χάρτη βασικών menu items του SPSS και βοήθεια για το εστιασμένο menu item κατά την πλοήγηση.
- Βελτιώθηκε η ανάγνωση πινάκων αποτελεσμάτων με μέγεθος πίνακα, πλουσιότερες συνόψεις γραμμής/στήλης και εντολή αντιγραφής της τρέχουσας γραμμής πίνακα.

## 1.0.0

Αρχική έκδοση.

- Προστέθηκε app module για IBM SPSS Statistics.
- Προστέθηκε αναγνώριση περιοχών για Output Viewer, Data View, Variable View, Syntax Editor και μενού.
- Προστέθηκαν λεπτομερείς περιγραφές περιοχών.
- Προστέθηκαν εντολές ανάγνωσης αποτελεσμάτων, κελιών δεδομένων και ορισμών μεταβλητών.
- Προστέθηκαν ετικέτες για συνηθισμένα κουμπιά και διαλόγους του SPSS.
- Προστέθηκαν aliases για συνηθισμένα ονόματα εκτελέσιμων αρχείων SPSS.
